# 我爱填单词 - 游戏图标设计方案

## 图标设计理念

### 核心元素
1. **填字游戏网格** - 代表游戏玩法
2. **英文字母** - 突出英语学习
3. **可爱萌系风格** - 吸引年轻用户群体
4. **明亮配色** - 活泼、积极的学习氛围

### 设计风格
- **萌系卡通风格**：圆润的线条，可爱的表情
- **扁平化设计**：适合应用图标展示
- **高对比度**：在各种背景下清晰可见
- **简洁有力**：小尺寸也能识别

## 配色方案建议

### 主色调
| 颜色 | 色值 | 用途 |
|------|------|------|
| 天空蓝 | #5BC0EB | 主背景、信任感 |
| 珊瑚粉 | #FF6B8A | 强调、活力 |
| 薄荷绿 | #7DCEA0 | 网格、正确反馈 |
| 奶油黄 | #FFE66D | 高亮、星星 |
| 纯白 | #FFFFFF | 字母、高光 |

### 渐变背景
- 浅蓝 → 粉色（#87CEEB → #FFB6C1）
- 薄荷绿 → 天蓝（#98FB98 → #87CEEB）
- 奶油黄 → 珊瑚粉（#FFFACD → #FFB6C1）

## 图标尺寸规格

| 平台 | 尺寸 | 格式 |
|------|------|------|
| iOS App Store | 1024×1024 | PNG |
| Android Play Store | 512×512 | PNG |
| iOS 主屏幕 | 180×180, 120×120, 60×60 | PNG |
| Android 自适应图标 | 432×432 (前景)，背景单独 | PNG |
| 网页 Favicon | 192×192, 32×32, 16×16 | PNG/ICO |

---

## 生成建议

1. **使用 Midjourney**：添加 `--v 6.1 --ar 1:1 --stylize 250` 获得最佳效果
2. **使用 DALL-E 3**：直接使用上述提示词，要求 1024x1024 尺寸
3. **使用 Stable Diffusion**：建议使用 Kawaii 或 Anime 相关模型

---

## 最终交付物

生成图标后，建议：
1. 导出为 PNG 格式，保留透明度
2. 在 Figma/Photoshop 中添加圆角蒙版
3. 按照各平台规格导出多尺寸版本
4. 保存原始高清版本用于宣传素材
