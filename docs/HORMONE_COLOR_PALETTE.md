# 荷尔蒙色（Hormone Color）配色方案

## 设计理念

荷尔蒙色（Hormone Color）是一种高饱和度、充满活力和情感张力的配色风格，旨在从柔和的马卡龙色转向更具视觉冲击力的配色方案。这种配色能够激发用户的情感共鸣，营造充满活力和激情的视觉体验。

### 核心特点
- **高饱和度**：使用100%饱和度的纯正色彩
- **强对比度**：通过高对比度营造视觉冲击
- **情感张力**：色彩充满活力和激情
- **现代感**：符合2024-2025年设计趋势

## 完整配色方案

### 1. 核心配色（替换马卡龙色）

#### 主色（Primary）- 活力珊瑚粉
- **主色**: `#FF4081` - 活力珊瑚粉（Vibrant Coral Pink）
  - 使用场景：主按钮、重要元素、强调色
  - RGB: (255, 64, 129)
  
- **深主色**: `#E91E63` - 深粉红（Deep Pink）
  - 使用场景：按钮阴影、边框、悬停状态
  - RGB: (233, 30, 99)
  
- **浅主色**: `#FF80AB` - 浅珊瑚粉（Light Coral Pink）
  - 使用场景：背景渐变、浅色按钮
  - RGB: (255, 128, 171)

#### 次要色（Secondary）- 电光蓝
- **次要色**: `#00E5FF` - 电光蓝（Electric Cyan）
  - 使用场景：次要按钮、信息提示、装饰元素
  - RGB: (0, 229, 255)
  
- **深次要色**: `#00BCD4` - 深青色（Deep Cyan）
  - 使用场景：按钮阴影、边框
  - RGB: (0, 188, 212)
  
- **浅次要色**: `#80D8FF` - 浅青色（Light Cyan）
  - 使用场景：背景渐变、浅色元素
  - RGB: (128, 216, 255)

#### 强调色（Accent）- 霓虹紫
- **强调色**: `#9C27B0` - 霓虹紫（Neon Purple）

  - RGB: (156, 39, 176)
  
- **深强调色**: `#7B1FA2` - 深紫色（Deep Purple）
  - 使用场景：阴影、边框
  - RGB: (123, 31, 162)
  
- **浅强调色**: `#CE93D8` - 浅紫色（Light Purple）
  - 使用场景：背景、浅色元素
  - RGB: (206, 147, 216)

#### 装饰色1 - 活力橙

  - RGB: (255, 183, 77)

#### 装饰色2 - 霓虹绿

#### 装饰色3 - 电光黄

### 4. 阴影和圆角定义

#### 阴影（Shadows）
- **软阴影**: `0 6px 0 rgba(255, 64, 129, 0.8)` - 主色阴影
- **3D阴影**: `0 4px 0 rgba(233, 30, 99, 0.9)` - 深主色阴影
- **卡片阴影**: `0 8px 16px rgba(0, 0, 0, 0.4)` - 深色卡片阴影
- **悬停阴影**: `0 12px 24px rgba(255, 64, 129, 0.5)` - 悬停效果
- **内阴影**: `inset 0 2px 4px rgba(0, 0, 0, 0.3)` - 内凹效果

#### 圆角（Border Radius）
- **大圆角**: `24px` - 卡片、大按钮
- **中圆角**: `16px` - 普通按钮、输入框
- **小圆角**: `12px` - 标签、小元素
- **超大圆角**: `32px` - 特殊装饰元素

## 配色对比表

| 元素 | 马卡龙色（旧） | 荷尔蒙色（新） | 变化说明 |
|------|---------------|---------------|---------|
| 主色 | #FFB6C1 (婴儿粉) | #FF4081 (活力珊瑚粉) | 更高饱和度，更有冲击力 |
| 背景 | #FFFAF0 (奶白色) | #1A1A2E (深蓝黑) | 从浅色转为深色，增强对比 |
| 文字 | #5D5D5D (深灰) | #FFFFFF (纯白) | 适配深色背景 |
| 次要色 | #87CEEB (天蓝) | #00E5FF (电光蓝) | 更高饱和度，更亮 |
| 强调色 | #FF69B4 (深粉) | #9C27B0 (霓虹紫) | 更强烈的视觉冲击 |

## 设计建议

1. **背景渐变**：使用深色背景配合霓虹色渐变，营造科技感和活力感
2. **对比度**：确保文字与背景有足够的对比度（WCAG AA标准）
3. **动画效果**：配合高饱和度颜色使用发光、闪烁等动画效果
4. **适度使用**：高饱和度颜色不宜大面积使用，应作为强调色
5. **三端一致性**：确保网页版、小程序、iOS应用使用相同的配色方案
